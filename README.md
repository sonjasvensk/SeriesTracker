# SeriesTracker

SeriesTracker is a Spring Boot CRUD web application for tracking TV series and writing reviews.

## Features
- User registration and authentication
- 1-5 star rating system
- Optional written reviews
- Role-based access control (USER / ADMIN)
- REST API support

## Tech Stack
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Thymeleaf
- Bean Validation

## Running the Application

Environment variables required:
- DB_URL
- DB_USERNAME
- DB_PASSWORD

Run with:
mvn spring-boot:run
